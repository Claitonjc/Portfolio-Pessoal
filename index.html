Projetos Pessoais

Vence Menos

Esse projeto é uma aplicação web para gestão de datas de validade de produtos. Foi criado para ser usado por exemplo em supermercados para o controle das validades dos produtos comercializados. Com essa aplicação podemos adicionar, remover e editar produtos e suas datas de validades, e também podemos configurar para receber alertas de validades próximas, ou definir qual o período escolhido para recolhimento dos produtos. Esse projeto foi feito a partir de um trabalho de faculdade, onde fizemos o design utilizando o Figma, e depois foi codificado.
